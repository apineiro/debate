Maitripa College Tibetan Program Digital Debate Site

This website is designed to assist students of Tibetan logic and debate. Currently it has these features:

  • A clickable list to the left of each page displaying some important terms (mtshon bya), definitions (mtshan nyid), divisions (dbye ba), and illustrations (mtshan gzhi) from Yong-dzin Pur-bu-jok's "Collected Topics: The Smaller Path of Reasoning".
  • A page displaying the possibilities for the comparison of phenomena.
  • An interactive page demonstrating the modes of a syllogistic reason.
  • An interactive page demonstrating correct and incorrect entailments.
  • An interactive page illustrating how to fling an absurd consequence in the presence of an incorrect entailment statement.
